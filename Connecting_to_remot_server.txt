1. Подключиться к удаленному серверу.

  ssh user_50@23.88.52.139

2. Посмотреть список папки. 

  ls -la

3.  Загрузить файл (p_l_s.py) на внешний сервер.

  scp user_50@23.88.52.139 :/home /user_50/p_l_s.py

4. Запустить скрин (для работы скрипта после отключения от сервера) 

  screen -s logs

5. Посмотреть список скринов

  screen -ls

6. Выйти из скрина.
 
  Ctrl+A+D

7. Зайти в скрин.

  screen -r 1123445 (id screen)

8. Запустить скрипт.

  python3 p_l_s.py
